# 第一章：太初之时，有设计师

游戏设计师，最重要的技能是“倾听”。

# 第二章：设计师创造体验。

游戏设计师真正关注的，是人们在玩游戏时产生的“体验"。我们所见的，所做的，所感受的，都是一种体验。

# 反思

设计游戏时，要善于反思自己的体验。但是自己的体验不一定适用于所有人。但是倾听太多，有可能就会把原本可以成为白天鹅的丑小鸭丢掉。所以，这是一项需要不断练习的技能。

当你玩一个游戏的时候，你必须分析他让你感受如何，它让你想到什么，他让你做了什么。你必须能够清晰地表达出你的分析。你必须用语言描述他。因为感觉是抽象的，而文字是具体的。你分析的越多，你就能越清晰地回到你的游戏应该创造哪种体验。

# 1号透镜：情感

人们可能会忘记你所说的，但他们永远不会忘记你带给他们的感觉         ---玛雅安吉罗

为了确定你创造的情感是正确的，问自己这几个问题：

* 我希望玩家能够体验到怎样的情感，为什么？

* 当人们（包括我自己）玩游戏时，他们产生了怎样的情感？为什么？

* 我怎样缩小实际体验与我的设想之间的差距？

---

但是反思本身就是一个巨大的挑战，观察行为本身就是一种体验，那么我们怎样才能获得不受干扰的体验呢？我们常常会面临这个问题，当你观察自己打字的时候，你会发现你的打字速度变慢了，同时发生了许多错误。这也被称为“海森堡原理”。那么，如何克服海森堡原理呢？

## 分析记忆

体验可以被我们记住，这是个好消息。但是我们很难在体验发生时分析体验。分析你记忆中的体验更加简单。记忆并不完美，但分析记忆总比什么都没有好。**不要刻意抱着分析的目的玩游戏，而是等到游戏结束之后，立刻分析这段玩游戏的记忆。带着这样的目的能够帮助你记得更多体验的细节，而不会干涉体验本身。**

## 两次经历

第一遍体验，第二遍分析。

# 本质体验

但是，怎么样才能让这些谈到的体验和观察结果真正融入游戏之中？**关键**就是，一个优秀的游戏不需要你完美复制真实的体验，你需要做的时为你的游戏找出这种**体验的本质。**那什么是“体验的本质”呢？每个值得纪念的体验都有一些让它与众不同的关键特征。

比如当你重温记忆中的打雪仗体验时，你可能会想起一大堆东西。其中一些你会认为是对于这种体验必不可少的：“雪太大了，学校都放假了。”“我们在街道上玩耍。”“积雪十分适合揉成雪球”“天气很冷，但是阳光充足——有蔚蓝色的天空。”“外面到处都是孩子。”“我们用雪堆了一个大堡垒。”“弗雷德的雪球扔得真高——我向上看时，他又扔了一个正好砸中我的头。”“我们情不自禁地笑了。”，而有一部分体验你会认为并不重要：“我穿着灯芯绒制成的裤子。”“我的口袋中有一些薄荷糖。”“一个通狗的人在看着我们。”

作为游戏设计师，在尝试设计一种体验时，你的目标是寻找能够定义你所想要的体验的基本元素，并想办法让它们成为你的游戏设计中的一部分。还是以打雪仗为例，你能用几种方式通关打雪仗的游戏表达“天气很冷”的体验呢？比如说，人物呼吸时会冒出白雾，动作会微微颤抖，用一些一道道呼啸而来的风带来冰冷的感觉，等等。

# 2号透镜：本质体验

要想使用这个透镜，你需要停止思考你的游戏，而开始思考玩家的体验。问自己这几个问题：

* 我想要让玩家获得怎样的体验？

* 这种体验的本质是什么？

* 我的游戏如何抓住这种本质？

# 第三章：体验发生于场景

场景，即游戏发生的地方，游戏可能发生在各个场景。考虑自己的游戏发生在哪个场景是非常重要的

### 壁炉边

人看着火就感觉到宁静，有安全感。而现代电视机就代替了壁炉的位置。壁炉边的游戏通常适合与家庭成员、朋友一起玩。或者能够随时被打扰的那种游戏类型

### 工作台

不适合在公开场合游玩，但是又有一定紧张感和私密性的游戏，如《英雄联盟》等

### 读书角

适合放松，安静地游玩的游戏，因为工作台坐着不舒服，而壁炉边又不私密，吵闹。

### 竞技场

游戏赛事，直播等。

### 游戏桌

桌游，面对面游戏（线下火影单挑hhh

随时随地

大部分手机游戏

# 3号透镜：场景

在我们的游戏设计中，玩游戏的场景会对游戏产生巨大的影响。为了确定你没有凭空设计游戏，问自己这几个问题：

* 我想要制作的游戏最适合哪种场景？

* 我的场景中是否有一些特殊的属性会影响我的游戏？

* 我的游戏中有哪些元素能够与场景和谐共处？哪些元素不行？
